# Day of Week Calculator (Doomsday Rule)

Консольная утилита на чистом C для быстрого и безопасного определения дня недели по любой дате в диапазоне от **01.01.0001** до **31.12.9999**.

В основе программы лежит **алгоритм Судного дня (Doomsday rule)** Джона Конвея. Это позволяет вычислять день недели за константное время O(1) без использования тяжелых системных библиотек работы со временем.

---

## 🌟 Ключевые особенности

* **Двуязычный интерфейс:** Полная поддержка русского и английского языков прямо в консоли.
* **Безопасность (Security-first):** Защита от уязвимостей форматной строки (вместо `printf(msg)` используется `printf("%s", msg)`).
* **Надежная валидация:** Корректно обрабатываются особенности високосных годов и диапазоны дней/месяцев.
* **Чистый код:** Код разбит на функции с единственной ответственностью (Single Responsibility) и документирован.

## 🛠️ Как это работает

Алгоритм Судного дня опирается на то, что определенные даты в году всегда приходятся на один и тот же день недели («Судный день» для этого года):

1. Вычисляется базовый день для столетия.
2. Вычисляется сдвиг внутри конкретного года в столетии.
3. Находится расстояние от целевой даты до ближайшего опорного дня текущего месяца.
4. Все компоненты складываются по модулю 7, давая точный день недели.
